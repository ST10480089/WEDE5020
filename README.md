# My Website Project 

## Overview 
This project is a fully interactive website with animations, a lightbox gallery, and dynamic content.

## Features
- Smooth JavaScript animations for text and images
- Responsive design
- SEO-ready with sitemap.xml and robots.txt
- Interactive gallery with lightbox

## Setup
1. Clone the repo: git clone https://github.com/yourusername/yourrepo.git
2. Open index.html in your browser
3. Ensure sitemap.xml and robots.txt are in the root folder
